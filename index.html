Apresentando ng-app, ng-model e ng-bind

Primeiro exemplo utilizando as três diretivas básicas do AngularJS.

Diretivas

A diretiva ng-app indica que a página, ou um bloco específico dela, é uma aplicação Angular.
Já a diretiva ng-model vincula o valor de um controle HTML - (input, select, textarea) - a dados do aplicativo.
A diretiva ng-bind vicula dados da aplicação ao HTML propriamente dito - também chamado de HTML View.

Algumas diretivas

Vimos que as diretivas AngularJS são atributos HTML com o prefixo ng.

A diretiva ng-init é responsável por "iniciar" variáveis na aplicação, conforme exemplo abaixo:

<div ng-init="primeiroNome='Felipe'">
    <p>he name is <span ng-bind="primeiroNome"></span></p>
</div>

Resultado:
The name is

Alternativa com HTML válido

<div data-ng-init="primeiroNome='Felipe'">
    <p>he name is <span data-ng-bind="primeiroNome"></span></p>
</div>

Resultado igual ao exemplo acima, porém com marcação válida de acordo com as web standards:
The name is

Você pode usar data-ng ao invés de ng- se quiser validar sua marcação HTML.

Expressões AngularJS

Expressões AngularJS são escritas entre chaves duplas: {{expressao}}

O AngularJS fará o "output" do dado exatamente onde a expressão está escrita, conforme e exemplo abaixo:

Minha primeira expressão: {{5 + 5}}

Resultado da minha primeira expressão: {{ 5 + 5 }}

Expressões AngularJS vinculam dados ao HTML da mesma maneira que a diretiva ng-bind.
