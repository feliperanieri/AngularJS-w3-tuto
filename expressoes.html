Expressões Angular JS

Em AngularJS, dados são vinculados ao HTML usando expressões.

Expressões Angular podem ser escritas entre chaves duplas - {{expression}} - ou, ainda, informadas através de uma diretiva - ng-bind="expression".
A expressão será interpretada e o resultado será informado exatamente onde ela foi escrita.

Expressões AngularJS são muito parecidas com expressões em Javascript puro, podendo contar literals, operadores ou variáveis.

Exemplo:
{{5 + 5}} ou {{primeiroNome + " " + ultimoNome}}

Exemplos práticos

                <div ng-app="">
                    <p>Minha primeira expressão: {{ 5 + 5 }}</p>
                </div>

Ver exemplo em funcionamento

É possível escrever uma expressão como quiser, o AngularJS sempre a interpretará e retornar o resultado.
Exemplo: Mudaremos o valor de uma propriedade CSS a qual alterará a cor de fundo do input abaixo de acordo com o valor digitado: 

                <div ng-app="" ng-init="myCol='lightblue'"</p>
                    <input style="background-color:{{myCol}}" ng-model="myCol" value="{{myCol}}">
                </div>

Trabalhando com números usando AngularJS

Podemos trabalhar com números em AngularJS de maneira similar ao Javascript convencional:

                <div ng-app="" ng-init="quantity=3;cost=5"</p>
                    <p>Total total em dólar: {{ quantity * cost }}</p>
                </div>
