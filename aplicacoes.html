Aplicações AngularJS

Em AngularJS modules definem as aplicações, os controllers, como o próprio nome diz, os controles na aplicação Angular.

No exemplo abaixo, a diretiva ng-app define a aplicação, ou seja, será nosso angular module e a diretiva ng-controller define o controle.

                    
<div ng-app="myApp" ng-controller="myCtrl">

    First Name: 

    Last Name: 

    

    Full Name: {{firstName + " " + lastName}}
    
        <script>
            var app = angular.module('myApp', []);
            app.controller('myCtrl', function($scope) {
                $scope.firstName= "John";
                $scope.lastName= "Doe";
            });
        </script>
    
</div>

Explicando o exemplo anterior

Módulo AngularJS definindo a aplicação:

var app = angular.module('myApp', []);

Informando controller para aplicação:
